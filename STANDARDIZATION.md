# Bitzup Standardization Cheatsheet

## 🔠 Typography Scale
| Token | Class | Size |
| :--- | :--- | :--- |
| **Hero** | `text-3xl` | 48px |
| **Section** | `text-2xl` | 32px |
| **Header** | `text-xl` | 24px |
| **Accent** | `text-lg` | 20px |
| **Main** | `text-base` | 16px |
| **Label** | `text-sm` | 14px |
| **Caption** | `text-xs` | 12px |

## 🖱️ Button Components
- **Primary** (`.btn-primary`): Red Gradient | White Text | 16px Rounded
- **Secondary** (`.btn-secondary`): White Box | Dark Text | 1.5px Gray Border

## 🎨 Design Tokens
- **Brand**: Teal (`#2EDBAD`) & Red Gradient (`#e91e63`→`#f44336`)
- **Theme**: Dark Mode (`#0B0E11` Background | `#1E2329` Surface)
- **Rounding**: Global `16px` (`rounded-2xl`)

## 📐 Layout Rules
- **Margins**: Use standard increments (`p-4`, `p-10`, `mt-20`).
- **Responsive**: Scale fonts with `md:` prefix (e.g., `text-xl md:text-3xl`).
